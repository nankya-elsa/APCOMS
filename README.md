# APCOMS

## AI-Powered Passenger Counting and Real-Time Occupancy Monitoring System for the Driverless Campus E-shuttle

### Project Overview

APCOMS is a system that combines edge computing and mobile application
technologies to provide real-time passenger counting and occupancy
monitoring for a driverless campus e-shuttle.

- mobile_app/ - Flutter-based occupancy display application
- database/ - Shared Firebase configuration
